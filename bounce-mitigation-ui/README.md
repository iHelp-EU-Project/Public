## Local Installation 

- Directly
```bash
pip install -r requirements.txt

streamlit run Start_Page.py
```

- With docker
    - Build the image
    ```bash
    docker build -t <image_tag> .
    ```
    - Run the image
    ```bash
    docker run -p 8050:8050 <image_tag>
    ```
